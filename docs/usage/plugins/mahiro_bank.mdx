### 说明

小真寻银行，提供高品质的存款！当好感度等级达到指初识时，小真寻会偷偷的帮助你哦。

- 插件类型: `NORMAL`
- 响应规则: `全局`
- 插件等级: `5`

### 配置项

- `sign_max_deposit`
  - 类型: `int`
  - 默认值: `100`
  - 说明: 好感度换算存款金额比例，当值是100时，最大存款金额=好感度*100，存款的最低金额是100（强制）
- `max_daily_deposit_count`
  - 类型: `int`
  - 默认值: `3`
  - 说明: 每日最大存款次数
- `rate_range`
  - 类型: `list[float]`    
  - 默认值: `[0.0005, 0.001]`
  - 说明: 小时利率范围
- `impression_event`
  - 类型: `int`
  - 默认值: `25`
  - 说明: 到达指定好感度时随机提高或降低利率
- `impression_event_range`
  - 类型: `list[float]`
  - 默认值: `[0.00001, 0.0003]`
  - 说明: 到达指定好感度时随机提高或降低利率
- `impression_event_prop`
  - 类型: `float`
  - 默认值: `0.3`
  - 说明: 到达指定好感度时随机提高或降低利率触发概率
  
  
### 指令

- 存款 [金额]
- 取款 [金额]
- 银行信息
- 我的银行信息
