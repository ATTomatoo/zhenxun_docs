### 说明

群事件处理

- 插件类型: `HIDDEN`
- 响应规则: `None`
- 插件等级: `None`

### 配置项

- `invite_manager`
  - `MESSAGE`:
    - 说明: 强制拉群后进群回复的内容
    - 类型: `str`
    - 默认值: `请不要未经同意就拉小真寻入群！告辞！`
  - `flag`
    - 说明: 是否强制拉群后进群退出并回复内容
    - 类型: `bool`
    - 默认值: `True`
  - `WELCOME_MSG_CD`
    - 说明: 群欢迎消息cd
    - 类型: `int`
    - 默认值: `5`


### 指令

无
