### 说明

执行 sql 语句与查看表

- 插件类型: `SUPERUSER`
- 响应规则: `全局`
- 插件等级: `5`

### 配置项

无

### 指令

- 查看所有表

- exec [sql]
  - 说明: 执行 sql 语句
