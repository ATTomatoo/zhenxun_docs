### 说明

早晚安被动技能

- 插件类型: `HIDDEN`
- 响应规则: `None`
- 插件等级: `None`
- 被动模块: `morning_goodnight`

### 配置项

无

### 指令

无

### 定时任务

- `morning`

  - 时间: 6:01
  - 任务: 广播早安

- `goodnight`
  - 时间: 23:59
  - 任务: 广播晚安
