title: 群管理员变动监测(admin_watch)
subSidebar: false
---

### 说明

检测群管理员变动, 添加与删除管理员默认权限,  
当配置项 ADMIN_DEFAULT_AUTH 为空时, 不会添加管理员权限

- 插件类型: `HIDDEN`
- 响应规则: `None`
- 插件等级: `None`

### 配置项

- admin_bot_manage
    - ADMIN_DEFAULT_AUTH
      - 说明: 设置群组管理员默认权限等级
      - 类型: `int`
      - 默认值: `5`

### 指令

无
